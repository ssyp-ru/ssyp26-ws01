# Calc — Examples

## Comments

```
// This is a comment
print 1 + 1; // trailing comment
```

## Arithmetic

```
print 1 + 2;
print 10 - 4;
print 2 * 3.5;
print 7 / 2;
print -5;
print -(3 + 4);
```

## Operator precedence and grouping

```
print 1 + 2 * 3;       // 7.000000
print (1 + 2) * 3;      // 9.000000
print 2 * 3 + 4 * 5;    // 26.000000
print (2 + 3) * (4 - 1); // 15.000000
```

## Booleans

```
assert true;
assert false;
assert !true;
assert !!true;
```

## Comparisons

```
assert 1 < 2;
assert 2 <= 2;
assert 3 > 5;
assert 3 >= 3;
assert 1 == 1;
assert 1 != 2;
assert (1 + 1) == 2;
```
